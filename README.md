# homebrew-tfenv

![merged] Merge to homebrew-core by @pyro2927 user.

Install the [tfenv](https://github.com/kamatama41/tfenv) via homebrew.

## Install
```bash
brew tap ringohub/tfenv
brew update && brew doctor
brew install tfenv
```
